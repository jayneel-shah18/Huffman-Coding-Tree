Huffman Coding Tree

What is Huffman Coding?

  • Huffman coding is a lossless data compression algorithm.
  • The idea is to assign variable-length codes to input characters, lengths of the assigned codes are based on the frequencies of corresponding characters.
  • The most frequent character gets the smallest code and the least frequent character gets the largest code.

What are the applications of Huffman Coding?

  • They are used for transmitting fax and text.
  • They are used by conventional compression formats like PKZIP, GZIP, etc.
  • Multimedia codecs like JPEG, PNG, and MP3 use Huffman encoding

Complexity Analysis of the code

  • The overall time complexity is O(nlogn).
  • We create a leaf node for each unique character and build a min heap of all leaf nodes.
  • Then we extract two nodes with the minimum frequency from the min heap.
  • Create a new internal node with a frequency equal to the sum of the two nodes frequencies. Make the first extracted node as its left child and the other extracted node as its right child and add this node to the min heap.
  • Repeat this till only one node is left.
  • For n nodes the time to extract the minimum node is logn and as it is called 2*(n-1) times, complexity is nlogn.
